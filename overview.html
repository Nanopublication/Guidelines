This document lists the Nanopublication documents that are being drafted.

Document overview

The following documents are planned or being drafted (tentative):
  • Overview (this document)
  • The Nanopublication schema
    • The nschema model
      • including recommendations for assertion, provenance and publication information
    • Nanopublication schema extensions
    • Relation to the Research Object model and ISA
    • Use cases (nanopublication modelling issues)
  • Nanopublication templates
  • Nanopublication store specification
    • includes specification of a Nanopublication API and back-end
  • Nanopublication primer or 'how to make nanopublications?'
    • Proposed design patterns for assertions
  • Nanopublication tutorial
  • Examples of nanopublications
  • Examples of nanopublication applications (including GUIs)
